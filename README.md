# Networking Journey

This repo contains a collection of notes and a hands-on assignment from the networking module of my DevOps journey.

## Contents

```
networking-journey/
├── assignment/   → Domain + EC2 + DNS project (NGINX live on a custom domain)
└── notes/        → Topic notes from the module
```

## Assignment

Bought a domain, launched an EC2 instance running NGINX, and pointed the domain to it via a Route 53 A record. Full write-up with screenshots in [`assignment/`](./assignment).

## Notes

Covers the core topics from the module:

- OSI Model & TCP/IP Model
- Intro to Networking (IP/MAC addresses, ports & protocols, TCP vs UDP)
- DNS
- Routing
- Subnetting & CIDR
- Cloud Networking (VPCs, subnets, gateways)
- Troubleshooting (ping, traceroute, nslookup)
