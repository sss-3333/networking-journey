# Cloud Networking

## What is cloud networking / importance
- The practice of managing and configuring networks in cloud environments
- Ensures applications and services can commnuc8iate smoothly and efficiently
- Works with virtualised components using software unlike traditional networking (routers, switches)

## Key Components
VPC's (Virtual Private Clouds)
Subnets
Gateways

### VPCs (Virtual Private Clouds)
- VPCs (Virtual Private Clouds): a network in the cloud where your own IP address range can be defined, create own subnets, configure route tables and gateway. Like setting up own data center in the cloud.

### Subnets
- Subdivisions of VPC
- Help organise network and control the flow of traffic

### Gateways
- What connects your VPC to the outside world like the internet
- Handle incoming and outgoing traffic, ensuring resources can communicate with external networks
